## Лабораторная работа №9
### Проверка супермножества

Условие:

Создать функцию superset(set1, set2), которая сравнивает множества и выводит одно из сообщений:

«Супермножество не обнаружено»
«Объект {X} является чистым супермножеством»
«Множества равны»

```python
def superset(a, b):

    if a == b:

        print("Множества равны")

    elif a > b:

        print("Объект A является чистым супермножеством")

    elif b > a:

        print("Объект B является чистым супермножеством")

    else:

        print("Супермножество не обнаружено")



set1 = {1, 2, 3}

set2 = {2, 3}

superset(set1, set2)

```
### Результат.
![Скриншот выполнения](pic/lab5_9.png)
## Выводы
Программа сравнивает два множества и сообщает, равны ли они, или одно является надмножеством другого.

Три списка:

```python
from math import sqrt



one = [12, 25, 3, 48, 71]

two = [5, 18, 40, 62, 98]

three = [4, 21, 37, 56, 84]



# Минимальные и максимальные стороны

a_min, b_min, c_min = min(one), min(two), min(three)

a_max, b_max, c_max = max(one), max(two), max(three)



def heron(a, b, c):

    p = (a + b + c) / 2

    return round(sqrt(p * (p - a) * (p - b) * (p - c)), 2)



print("Площадь треугольника из минимальных сторон:", heron(a_min, b_min, c_min))

print("Площадь треугольника из максимальных сторон:", heron(a_max, b_max, c_max))


```

## Самостоятельная работа №5
### Вам предоставлены списки натуральных чисел, из них необходимо сформировать множества. При этом следует соблюдать правило:если какое-либо число повторяется, то преобразовать его в строку по следующему образцу: например, если число 4 повторяется 3 раза, то в множестве будет следующая запись: само число 4, строка "44", строка "444".

Множества для теста:

list_1 = [1, 1, 3, 3, 1]

list_2 = [5, 5, 5, 5, 5, 5, 5]

list_3 = [2, 2, 1, 2, 2, 5, 6, 7, 1, 3, 2, 2]

Результаты вывода (порядок может отличаться, поскольку используется set()):

{'11', 1, 3, '33', '111'}

{5, '5555', '555555', '55555', '555', '55', '5555555'}

{'11', 1, 3, 2, 5, 6, '222222', '222', 7, '2222', '22222', '22'}

```Python
def create_custom_set(numbers):

    result = set()

    for num in set(numbers):

        count = numbers.count(num)

        for i in range(1, count + 1):

            if i == 1:

                result.add(num)

            else:

                result.add(str(num) * i)

    return result



list_1 = [1, 1, 3, 3, 1]

list_2 = [5, 5, 5, 5, 5, 5, 5]

list_3 = [2, 2, 1, 2, 2, 5, 6, 7, 1, 3, 2, 2]



print(create_custom_set(list_1))

print(create_custom_set(list_2))

print(create_custom_set(list_3))


```

## Общие выводы по теме
Списки (list):

Позволяют хранить набор данных в определённом порядке
Поддерживают изменение элементов (добавление, удаление, замена)
Можно сортировать, искать минимальные и максимальные значения
Удобны для работы с однотипными данными (оценки, результаты, коды работников)


Множества (set):

Хранят только уникальные значения
Автоматически исключают дубликаты
Поддерживают математические операции (объединение, пересечение, разность)
Используются, когда важен факт наличия элемента, а не его порядок


Работа с коллекциями:

Позволяет выполнять анализ данных (подсчёт, фильтрация, поиск максимумов и минимумов)
Упрощает реализацию задач с повторяющимися элементами
Требует понимания различий между изменяемыми и неизменяемыми структурами


Архитектура программ:

Использование циклов и функций для автоматизации повторяющихся действий
Разделение кода на логические блоки повышает читаемость
Применение стандартных модулей (например, math) делает код универсальнее


Итог:

Освоение базовых коллекций Python (списков и множеств) позволяет эффективно хранить, обрабатывать и анализировать данные, что является фундаментом для построения более сложных программных решений.
